[https://www.acmicpc.net/problem/12865](https://www.acmicpc.net/problem/12865)
```javascript
const input = require('fs')
    .readFileSync(process.platform === 'linux' ? '/dev/stdin' : './input.txt')
    .toString()
    .trim()
    .split('\n');

let inputIndex = 0;

// N개의 물건, 버틸 수 있는 배낭의 무게 K
const [N, K] = input[inputIndex++].split(' ').map(Number);

// dp[i][j] = 처음부터 i번째 물건까지 고려했을 때, 배낭의 무게 한도가 j일 때의 최대 가치
const dp = Array.from({ length: N + 1 }, () => Array(K + 1).fill(0));

// 1개의 물건부터 N개의 물건까지 확인한다.
for (let i = 1; i <= N; i++) {
    // 지금 물건
    const [W, V] = input[inputIndex++].split(' ').map(Number);

    // 가방 크기만큼 순회
    for (let j = 1; j <= K; j++) {
        // 이 물건을 집었을 때. W무게만큼 증가해야 하므로, j-W 열 값을 가져온다.
        let case1;
        // 이 물건을 안 집었을 때. 이전 물건까지 집었을 때, j무게 가치를 가져온다.
        let case2;

        // 이 무게를 가방에 넣을 수 있다면
        if (j - W >= 0) {
            case1 = dp[i - 1][j - W] + V;
            case2 = dp[i - 1][j];

            // 두 경우 중 더 큰 값을 저장한다.
            dp[i][j] = Math.max(case1, case2);
        } else {
            // 이 무게를 가방에 넣을 수 없다면 물건을 집지 못한다.
            case2 = dp[i - 1][j];
            dp[i][j] = case2;
        }
    }
}

console.log(dp[N][K]);
```
### 🔗 풀이
1. 🤔 테이블 정의하기
    1. `dp[i][j] = 처음부터 i번째 물건까지 고려했을 때, 배낭의 무게 한도가 j일 때의 최대 가치`
    2. `i` = 지금 넣을지 말지 보고있는 물건
    3. `j` = 현재 배낭의 허용 무게를 1부터 K까지 순환해본다.

2. 🤔 점화식 찾기
    1. `W`: 물건의 무게, `V`: 가치
    2. `dp[i][j] = max(dp[i - 1][j - W] + V, dp [i-1][j])` 이다.
    3. 여기서 `dp[i-1][j-w] + V`가 핵심이다. `w`무게만큼 앞(왼쪽)에 있는 열(`j-W`)의 가치와 가치(`V`)를 더해서 바로 윗행 `dp[i-1][j]`와 비교하여 최대 가치를 가져온다.

3. 🤔 초기값 정하기
    1. 아무 물건도 고려하지 않은 상태(0행)와 배낭 용량이 0인 상태(0열)는 모두 가치가 0이므로 0으로 초기화한다.
### 🔗 배운점
1. 두 번째 `for`문 최적화
   - `for (let j = W; j <= K; j++)` 애초에 `W`무게부터 허용한다고 시작하면 조건문을 굳이 안써도 된다.
