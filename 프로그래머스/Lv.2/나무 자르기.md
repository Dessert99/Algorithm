[https://www.acmicpc.net/problem/2805](https://www.acmicpc.net/problem/2805)
```javascript
const input = require('fs')
    .readFileSync(process.platform === 'linux' ? '/dev/stdin' : './input.txt')
    .toString()
    .trim()
    .split('\n');

// N: 나무 수, M: 필요한 나무 길이
const [N, M] = input[0].split(' ').map(Number);

// 나무 한 줄
const trees = input[1].split(' ').map(Number);

// 정답
let answer = 0;

// 절단기 높이 범위
let start = 0;
let end = 1000000000;

// 이분 탐색 시작. 절단기 높이를 조절하면서 최댓값을 구한다.
while (start <= end) {
    // 현재 절단기 높이
    const mid = Math.floor((start + end) / 2);

    // 잘린 나무 누적값
    let totalCutDownTrees = 0;

    // 현재 절단기 높이로 나무 잘라보기
    for (let i = 0; i < trees.length; i++) {
        // 잘린 나무 길이
        let cutDownTree = trees[i] - mid;
        // 나무가 절단기 높이보다 길어야 한다.
        if (cutDownTree >= 0) {
            totalCutDownTrees += cutDownTree;
        }
    }

    // 만약 잘린 나무 누적값이 필요한 나무(M)길이 이상이라면 조건을 만족한다.
    if (totalCutDownTrees >= M) {
        answer = mid; // 일단 정답으로 가정한다.

        // 절단기 길이를 늘려서 다시 조사한다.
        start = mid + 1;
    } else {
        // 조건을 만족하지 못했으니, 절단기 길이를 줄여서 다시 조사한다.
        end = mid - 1;
    }
}

console.log(answer);
```
### 🔗 풀이
1. 이분 탐색 범위를 정한다. 절단기의 높이이다.
    - `start`: 0
    - `end`: 1,000,000,000
2. 이분 탐색을 시작한다. 
3. 중간값 `mid`를 구하고, `mid`길이로 나무를 잘랐을 때, 필요한 나무 길이(`M`)을 만족하는지 확인한다.
4. 현재 `mid`가 필요한 나무 길이를 만족하는지 `true`/`false`값을 반환하는 함수가 필요하다.
    - 함수는 현재 절단기 길이(`mid`)와 나무 배열, 필요한 나무 길이(`M`)을 매개변수로 받는다.
    - 잘린 나무 길이를 저장할 `totalTreeLength` 변수를 선언한다.
    - 나무 배열을 순회하며 `mid`길이만큼 뺀 나머지를 `totalTreeLength`변수에 더한다.
    - `totalTreeLength` 변수와 `M`을 비교하여, 요구조건을 만족했으면 `true`, 하지 못했다면 `false`를 반환한다.
5. 만약 함수가 `true`를 반환하면 필요한 나무를 충족하므로 `answer`에 저장한다. 하지만 더 높은 절단기가 조건을 만족할 수 있으므로, `start = mid +1` 로 수정하고 이분 탐색을 다시 실행한다.
6. 만약 함수가 `false`를 반환하면 필요한 나무를 충족하지 못한다는 뜻이다. 절단기 높이가 너무 높아서 필요한 만큼 나무를 자르지 못한 것이니, `end = mid -1` 로 수정하여 이분 탐색을 다시 실행한다.

### 🔗 배운점
1. 이분 탐색 특징
   - "이분 탐색 문제 해결의 핵심은 '조절할 변수(Input)'와 '만족해야 할 조건(Output)'을 정의하는 것이다.
   - 이 문제에서는 '절단기의 높이'가 조절해야 할 변수(0 ~ 10억)이고, 이를 통해 '가져갈 나무의 총 길이'가 $M$ 이상이 되는지 확인하는 것이 조건이다."
