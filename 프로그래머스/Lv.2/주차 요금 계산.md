```javascript
// records: 1~1000
function solution(fees, records) {
    var answer = [];
    // T: 기본 시간(1~1439), W: 기본 요금(0~10만), 단위 시간(1~1439), 단위 요금(1~1만)
    const [T, W, unit_T, unit_W] = fees;

    // 입/출차 내역 길이
    const N = records.length;

    // 차량 입출차 해시맵
    const carMap = new Map();

    // 누적 시간 해시맵
    const sumMap = new Map();

    for (let i = 0; i < N; i++) {
        // 시각("09:59"), 차량번호("0948"), 내역("IN" "OUT"),
        const [time, carNum, state] = records[i].split(' ');
        // 처음 보는 차량은 sumMap에 넣는다
        if (!sumMap.has(carNum)) {
            sumMap.set(carNum, 0);
        }

        // 입차 차량이라면
        if (state === 'IN') {
            // 들어와
            carMap.set(carNum, time);
        } else {
            // 출차 차량이라면
            // 일단 입차 시간 꺼냄
            const inTime = carMap.get(carNum);

            // 추차된 시간을 구함
            const parkedMinute = calculateTime(inTime, time);

            // 주차 시간 누적
            sumMap.set(carNum, sumMap.get(carNum) + parkedMinute);

            // 해시맵에서 삭제
            carMap.delete(carNum);
        }
    }
    // 순회가 끝났는데 carMap에 입차된 차량이 있다면 23:59 출차
    for (const [key, value] of carMap) {
        const carNum = key;
        const time = value;

        //추차된 시간
        const parkedMinute = calculateTime(time, '23:59');

        // 누적
        sumMap.set(carNum, sumMap.get(carNum) + parkedMinute);

        // 해시맵에서 삭제
        carMap.delete(carNum);
    }

    // 요금 정산
    const sumArr = [];
    for (const [key, value] of sumMap) {
        const carNum = key;
        const totalTime = value;

        // 누적 시간이 기본 시간보다 작다면 기본 요금
        if (totalTime <= T) {
            sumArr.push([carNum, W]);
        } else {
            // 더 썼으면 계산
            const cal = W + Math.ceil((totalTime - T) / unit_T) * unit_W;
            sumArr.push([carNum, cal]);
        }
    }

    // 정렬: 0번 인덱스가 차량 번호이다.
    sumArr.sort((a, b) => a[0] - b[0]);

    // 이제 순서대로 요금을 answer에 넣는다.
    for (let i = 0; i < sumArr.length; i++) {
        answer.push(sumArr[i][1]);
    }

    // 출차, 입차 시간을 받아서 차이를 분 단위로 리턴하는 함수
    function calculateTime(inTime, outTime) {
        // '14:29' - '08:45' 예시
        // 입차 시, 분
        let [h1, m1] = inTime.split(':').map(Number);
        // 출차 시, 분
        let [h2, m2] = outTime.split(':').map(Number);

        // 시 -> 분
        m1 += h1 * 60;
        m2 += h2 * 60;

        return m2 - m1;
    }

    return answer;
}
```
### 🔗 풀이
💡 주의사항
    1. 출차된 내역이 없다면 23:59에 출차
    2. 출차하고, 또 입차할 수 있다.

💡 문제 조건
    1. 올림 -> Math.ceil()
    2. records는 시각 기준 오름차
    3. 차량 번호가 작은 자동차부터 청구할 주차 요금 출력

🤔 우리가 계산해야할 것.
    - 입차했을 때 시각, 출차했을 때 시각을 비교해야 한다.
    - 출차했을 때 로직을 계산해야 한다.
    

🤔 해시맵에 저장할까?
    - 차량 번호를 key값으로 한다면, 시각과 내역을 value로 묶어야 할까?
    - 👀 방법1)
        1. key: 차량 번호 value: {시각, 내역, 총 요금}
        2. 입차 시 맵에 저장
        3. 출차 시 맵에서 꺼낸 뒤 파싱, 계산 시작.
        4. 계산 끝나고 다시 맵에 총 요금 넣음.
### 🔗 배운점
1. `Map`
   - 요소 반환
     ```javascript
      const map = new Map();
      map.set('A', 100);
      
      // 1. map.has(key) : 존재 여부 
      console.log(map.has('A')); // true
      console.log(map.has('B')); // false
      
      // 2. map.get(key) : 값 반환
      console.log(map.get('A')); // 100 (값 반환)
      console.log(map.get('B')); // undefined (없으면 undefined)
     ```
   - `Map` 순회
     ```javascript
      const map = new Map([['A', 100], ['B', 200]]);
      // 1. for...of (구조 분해 할당 활용 )
      for (const [key, value] of map) {
          console.log(key, value); // 'A' 100
      }
     
      // 2. forEach (value가 먼저 옴 주의!)
      map.forEach((value, key) => {
          console.log(key, value);
      });
      
      // 3. 키만 혹은 값만 필요할 때
      for (const key of map.keys()) { ... }
      for (const val of map.values()) { ... }
     ```
  - `Map` → 배열 변환
    ```javascript
      const map = new Map([['apple', 3], ['banana', 5]]);
      
      // 1. [Key, Value] 쌍의 2차원 배열로 변환 (정렬 시 필수 ⭐)
      const arr = [...map]; 
      // 결과: [['apple', 3], ['banana', 5]]
      
      // 활용: 값 기준 내림차순 정렬
      arr.sort((a, b) => b[1] - a[1]);
      
      // 2. Key만 배열로
      const keys = [...map.keys()]; // ['apple', 'banana']
      
      // 3. Value만 배열로
      const values = [...map.values()]; // [3, 5]
    ```
2. `Set`: 중복을 자동으로 제거함. 값 자체가 Key이자 Value.
   - 요소 넣기 (add)
     ```javascript
        const set = new Set()
         set.add(2)
         set.add(2)
         set.add(3)
        console.log(set) // Set(2) { 2, 3 }
     ```
   - 크기 확인 (size)
     ```javascript
         const set = new Set()
         set.add(2)
         set.add(2)
         set.add(3)
         console.log(set.size) // 2
     ```
       - `size()`가 아니라, `size`이다.
   - 요소 확인 (has)
     ```javascript
      const set = new Set([1, 2, 3]);
      
      // 1. set.has(value) : 존재 여부 (Boolean)
      console.log(set.has(2)); // true
      console.log(set.has(5)); // false
      
      // ⚠️ 주의: Set은 인덱스로 접근 불가, get() 메서드 없음.
      // 값을 꺼내려면 순회하거나 배열로 바꿔야 함.
     ```
   - `Set` 순회
     ```javascript
      const set = new Set(['A', 'B', 'C']);
      
      // 1. for...of (추천 ⭐)
      for (const val of set) {
          console.log(val);
      }
      
      // 2. forEach
      set.forEach((val) => {
          console.log(val);
      });
     ```
   - `Set` → 배열 변환
     ```javascript
      const numbers = [1, 2, 2, 3, 3, 4];
      const set = new Set(numbers); // Set {1, 2, 3, 4}
      
      // 1. Spread 연산자 (가장 간단 ⭐)
      const uniqueArr = [...set]; // [1, 2, 3, 4]
      
      // 2. Array.from
      const uniqueArr2 = Array.from(set);
     ```
