```javascript
function solution(phone_book) {
    var answer = true;

    const map = new Map();

    // 배열 -> 해시맵에 저장
    for (const phone of phone_book) {
        map.set(phone, true);
    }

    // 이제 접두사 확인
    for (const phone of phone_book) {
        const n = phone.length;
        // n-1 까지 확인하는게 포인트이다. n까지 확인하면, 자기 자신을 찾게 된다.
        for (let i = 0; i < n - 1; i++) {
            const prefix = phone.slice(0, i + 1);
            if (map.has(prefix)) {
                answer = false;
            }
        }
    }
    return answer;
}
```
### 🔗 배운점
1. 시간복잡도
   - 반복문을 사용하기 전에 꼭 시간복잡도를 확인하자. 전화번호 목록은 100만, 각 번호의 최대 길이는 20이기 떄문에 2000만으로 충분한 연산이다.
2. 접두사 비교하기
   - 시간복잡도 문제가 해결되기 때문에 각 번호의 앞자리를 늘려가며 비교할 수 있다.
