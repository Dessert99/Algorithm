[https://leetcode.com/problems/biggest-single-number/description/](https://leetcode.com/problems/biggest-single-number/description/)
```sql
WITH
    result as (
        SELECT
            num,
            COUNT(num) OVER (PARTITION BY num) AS count_num
        FROM
           MyNumbers 
    )

SELECT
    -- MAX() 집계 함수는 비교할 행이 없다면 null을 반환한다.
    MAX(num) AS num
FROM
    result

WHERE
    -- 조건1) 유일한 num만 뽑기
    count_num = 1
```
### 🔗 사고 과정
1. 🤔 단일 숫자 파악
    - 어떻게? -> COUNT() OVER()로 새로운 컬럼을 만들어서 1인 행만 고른다.

2. 🤔 단일 숫자가 없으면 null

### 🔗 배운점
1. 윈도우 함수 풀이 해설
   1. CTE 테이블 만들기
   2. `ROW_NUMBER()` 윈도우 함수로 `num`이 몇 개 나왔는지 새로운 컬럼(`count_num`)을 추가한다.
   3. 본 쿼리에서 `count_num`이 1인 행만 필터링한다.
   4. 그 중에서 가장 큰 값만 `MAX()`으로 뽑는다.
2. 일반 풀이: `GROUP BY + HAVING`
   ```sql
      WITH
          result AS (
              SELECT
                  num
              FROM
                  MyNumbers
              GROUP BY
                  num
              HAVING
                  -- 그룹화 했을 때, 값이 하나인 행만 필터링
                  COUNT(num) = 1
          )
      
      SELECT
          MAX(num) AS num
      FROM
          result
   ```
   1. `num`기준 그룹화해서 뭉쳐진 데이터를 확인한다
   2. 만약 뭉쳐진 데이터가 여러 개라면 -> 중복된 값이다.
