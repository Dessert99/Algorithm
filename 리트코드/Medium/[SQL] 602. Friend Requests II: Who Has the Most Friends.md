```sql
WITH
    union_table AS (
        SELECT
            requester_id
        FROM
            RequestAccepted
        UNION ALL
        SELECT
            accepter_id
        FROM
            RequestAccepted
    )

SELECT
    requester_id AS id,
    COUNT(requester_id) AS num

FROM
    union_table

GROUP BY
    requester_id

-- 가장 많은 행을 구하고 싶으면 정렬하고 맨 위에서 가져온다.
ORDER BY
    num DESC
LIMIT 1
```
### 🔗 풀이

### 🔗 배운점
1. `UNION`을 쓰는 이유
   - 우리가 집계해야 하는 `id`는 두 컬럼에 섞여있다. `union`작업으로 하나의 컬럼에 모아놓은 뒤, 집계한다.
2. 가장 많은 하나의 행을 고르는 방법
   - 집계 순으로 정려하고, `LIMIT`문법으로 가장 상위 요소만 가져오면 된다.
