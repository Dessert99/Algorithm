[https://leetcode.com/problems/nth-highest-salary/](https://leetcode.com/problems/nth-highest-salary/)
```sql
CREATE FUNCTION getNthHighestSalary(N INT) RETURNS INT
BEGIN

  RETURN (
        WITH 
        RANKING AS (
            SELECT
                SALARY,
                DENSE_RANK() OVER (ORDER BY SALARY DESC) AS rnk
            FROM
                Employee
        )

        SELECT
            MAX(SALARY)
        FROM
            -- 여기에 랭킹이 들어간 테이블
            RANKING
        WHERE
            rnk = N
  );
END
```
### 🔗 풀이
1. 💡 문제에서 요구하는 것
    - 연봉을 높은 순으로 정렬했을 때, `N`번째 사람.
    - `ID`가 `N`인 사람이 아니다. 랭킹이 `N`번째 사람.
### 🔗 배운점
1. 사고 과정
   1. `N`에 어떤 숫자가 들어올지 모르니, 랭킹 순서가 적힌 새로운 컬럼을 만들어야 한다.
   2. 참고로 `DENSE_RANK` 함수는 등수를 건너뛰지 않는다. 공동 1등이 있다면, 다음 등수는 2등이다.
   3. 만약 `N`이 등수에 없다면 `NULL`을 반환해야 한다. 이를 위해 `MAX`집계 함수를 사용한다.
2. `MAX`집계 함수가 `NULL`을 반환해주는 이유
   - `MAX()`, `MIN()`, `SUM()` 같은 집계 함수는 조건에 맞는 데이터가 아예 없더라도 무조건 1행을 반환하려 한다. 이때 `NULL`을 반환한다.
   - 다만, `COUNT()`함수는 조건에 맞는 데이터가 없으면 0을 반환한다.
3. `IFNULL`을 쓰면 안되는 이유
   - `IFNULL`은 행이 비었을 때(그 행이 `NULL`일 떄), 특정 값으로 바꾸는 함수이다. 이 문제에서 행이 아예 없기 떄문에 `IFNULL()`은 적절하지 않다.
